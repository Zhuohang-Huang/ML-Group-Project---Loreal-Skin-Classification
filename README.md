# ML-Group-Project---Loreal-Skin-Classification
Multi-label skin condition classifier built for the L'Oréal Hackathon. Classifies 33 skin conditions from beauty product descriptions (6,240 samples) using TF-IDF + Ridge Classifier. Data-centric approach with label restructuring &amp; oversampling. F1=66% | CO₂ ≤ 0.100 kg tracked via CodeCarbon.
